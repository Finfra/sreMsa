## 5. 아래의 yml파일을 가지고 다시 생성하시오.
```
kubectl ~
cat <<EOF>u1.yml
apiVersion: v1
kind: Pod
metadata:
  name: u1
  labels:
    server: ubuntu
spec:
  containers:
  - image: ubuntu
    command:
      - "sleep"
      - "604800"
    imagePullPolicy: IfNotPresent
    name: u1
  restartPolicy: Always
EOF
kubectl ~
```
